## Summary
**Tic-Tac-Toe** – A fully interactive and responsive **web-based Tic-Tac-Toe game** built with **HTML, CSS, and JavaScript**. Features real-time turn management, win and draw detection, and a clean interface for desktop and mobile users.

## Overview
- Implemented **game logic** for player turns, win detection, and draw handling.  
- Designed a **responsive UI** compatible with desktop, tablet, and mobile devices.  
- Used **DOM manipulation** to update the game board in real time and handle user input.  
- Structured code for readability and easy maintenance.
